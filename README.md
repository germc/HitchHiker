# HitchHiker

Full-stack Uber clone using Firebase as a backend server.

<h1>Concept used</h1>

<ol>
<li>Location Manager</l1>
<li>MapKit- Overlay and Annotations</li>
<li>Firebase Authorization</li>
<li>Firebase Database</li>
<li>Custom classes for UI</li>
</ol>
